# RTDM

### Docker Cheat Sheet   
```sh
$ docker build -t netpoleon/rtdm .
$ docker images
$ docker run -p 49160:8080 -d netpoleon/rtdm
$ docker ps
```

Open Web Browser and go to
- [localhost:49160](http://localhost:49160/)
